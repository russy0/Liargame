# Liar Game Discord Bot

디스코드에서 라이어게임을 진행하는 봇입니다.

참가자 모집, 역할 배정 DM, 토론 타이머, 지목 투표, 라이어 최종 추측, 전적 기록을 처리합니다.

## 실행 방법

```bash
pip install -r requirements.txt
python bot.py
```

`.env` 파일에 봇 토큰을 넣어야 합니다.

```env
DISCORD_TOKEN=your_bot_token_here
```

## 주요 명령어

- `/라이어시작` 참가자 모집 후 게임 시작
- `/라이어중지` 진행 중인 게임 또는 모집 중지
- `/라이어설정` 기본 인원/시간/역할 설정
- `/라이어상태` 현재 게임 상태 확인
- `/라이어주제` 단어 카테고리 검색
- `/라이어규칙` 규칙 확인
- `/라이어내정보` 내 전적 확인
- `/라이어리더보드` 전적 순위 확인

## 규칙

시민은 같은 제시어를 받고, 라이어는 주제만 받습니다.
봇이 발언 순서를 정하고 참가자는 돌아가며 토론합니다.
토론 중 과반수 버튼 투표로 토론을 스킵하거나 시간을 연장할 수 있습니다.
토론 후 라이어로 의심되는 사람에게 투표합니다.
시민을 지목하거나 동률, 스킵, 무투표가 나오면 라이어가 승리합니다.
라이어를 지목하면 라이어가 제시어를 추측하고, 맞히면 라이어 승리, 틀리면 시민 승리입니다.

## 설정

기본 설정과 단어장은 `config.json`에서 관리합니다.
게임 중에는 `/라이어설정`으로 기본 라이어 수, 인원, 타이머, 역할 이름을 바꿀 수 있습니다.

봇 초대 시 `Server Members Intent`와 역할 관리 권한이 필요합니다.
